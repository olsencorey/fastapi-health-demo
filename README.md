# fastapi-health-demo
fastapi-health-demo is a Python microservice that exposes a health risk scoring API built with FastAPI. It accepts structured patient or health input and returns a computed risk score through a clean, documented REST endpoint.

The service is fully containerized with Docker and ships with a pytest test suite to verify scoring behavior. Auto-generated OpenAPI docs (via FastAPI's built-in Swagger UI) are available at /docs out of the box.

Tech Stack: Python · FastAPI · REST API · OpenAPI/Swagger · Docker · pytest
